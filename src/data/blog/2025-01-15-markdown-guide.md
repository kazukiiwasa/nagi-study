## テキストの装飾

通常の段落テキストです。段落と段落の間には適切なスペースが設けられます。

**太字のテキスト**は重要な情報を強調する際に使用します。

*イタリック体のテキスト*は、引用や用語を強調する際に使用します。

**_太字とイタリックを組み合わせる_**こともできます。

## コード

インラインコード: `const greeting = "Hello, World!";`

コードブロック:

```javascript
function calculateTotal(items) {
  return items.reduce((sum, item) => sum + item.price, 0);
}

const items = [
  { name: "商品A", price: 1000 },
  { name: "商品B", price: 2000 },
];

console.log(calculateTotal(items)); // 3000
```
